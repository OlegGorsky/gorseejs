# Gorsee AI Tools

Reads `.gorsee/ide/*` files generated by:

```bash
bunx gorsee ai ide-sync --watch
```

This extension works in VS Code and Cursor because both support the VS Code extension model.

## Features

- Publishes diagnostics from `.gorsee/ide/diagnostics.json`
- Watches `.gorsee/ide/context.md`
- Command: `Gorsee AI: Show Context`
- Command: `Gorsee AI: Refresh Diagnostics`

## Install

1. Stage the extension:

```bash
bun run ai:package:vscode
```

2. Install `dist/vscode-gorsee-ai/gorsee-ai-tools-0.1.0.vsix` via `Extensions: Install from VSIX...`, or open `dist/vscode-gorsee-ai/stage/extension` as an extension development host.
3. In your app workspace, run:

```bash
bunx gorsee ai ide-sync --watch
```

4. Open the project in VS Code or Cursor.

## Notes

- The extension does not execute your app.
- It only consumes generated AI projection files.
- If no diagnostics appear, confirm `.gorsee/ide/diagnostics.json` exists.
